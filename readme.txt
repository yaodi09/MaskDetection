Modules you might need:
    pip3 install tensorflow
    pip3 install pandas
    pip3 install pillow
    pip3 install opencv-python
    pip3 install scikit-learn

To run the code:
1. The trained model is already included, if you want to test the train code, switch to the project directory and type 'python3 train.py' in cmd
2. To test the code, switch to the project directory and type 'python3 mask.py' in cmd, you might need to give access for the camera on the first time, so the program might auto shut down, just run it again after grant the access to the camera
